# Security Policy

## Supported Versions

Security fixes are provided for the latest published versions of
`@proveit/verification-contract` and `@proveit/open-verifier`.

## Reporting a Vulnerability

Please report suspected vulnerabilities by emailing security@proveit-app.com.
Do not open a public issue for sensitive reports.

Include the affected package, version, reproduction steps, impact, and any
known mitigations. We will acknowledge receipt within 3 business days and follow
up with triage status once the issue has been assessed.

## Scope

This repository contains public verifier tooling and public verification record
contracts. Backend credentials, private infrastructure, production signing keys,
and application attestation secrets are out of scope for this repository and
must not be committed here.
